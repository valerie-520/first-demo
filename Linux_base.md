# 一、路径查看与切换（目录操作核心）
1. pwd
作用：打印当前所在绝对路径
示例：
pwd
 输出：/home/fishros/chapt1
2. cd 切换目录
cd /：切换到根目录（系统最顶层）
cd ~：切换到当前用户家目录（等价于直接输入 cd）
cd 文件夹名：进入当前目录下的指定文件夹，例 cd chapt1/
补充：cd .. 返回上一级目录
3. ls 列出目录内容
作用：查看当前文件夹内所有文件、子文件夹
 输出示例：公共的 模板 视频 chapt1 snap ...
拓展：ls -l 查看文件权限 / 大小 / 创建时间，ls -a 显示隐藏文件
# 二、文件夹创建：mkdir
作用：新建空目录
mkdir chapt1
 创建名为 chapt1 的文件夹
# 三、文件创建：touch
作用：新建空白文本文件（不会打开编辑，仅创建）
bash
运行
touch hello_world.txt
在当前目录生成 hello_world.txt 空文件
# 四、文本编辑工具：nano
作用：轻量简易终端编辑器，适合新手快速改文本
nano hello_world.txt
 打开文件进行编辑
基础快捷键（nano 内）：
Ctrl+O：保存修改
Ctrl+X：退出编辑器
Ctrl+W：搜索文本
# 五、目录结构基础认知
根目录 /：系统总根，存放系统核心文件夹
常见子目录：bin(系统命令)、etc(配置文件)、home(所有用户家目录)、tmp(临时文件)、usr(软件安装目录)
家目录 ~：每个用户独立工作目录，存放桌面、文档、自定义文件夹，权限隔离
# 六、命令cat
作用
读取并直接在终端打印文件全部内容，用于快速查看文本文件，无需打开编辑器。
示例
cat hello_world.txt
 输出文件内文字：hello ros 2!
拓展用法
cat 文件名1 文件名2：同时打印多个文件内容
cat > test.txt：新建文件并实时输入内容（Ctrl+D保存退出）
